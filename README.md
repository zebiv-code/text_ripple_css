# Text Ripple (CSS)

CSS-driven water ripple effect on a beautifully typeset page of *Alice's Adventures in Wonderland*. Move your mouse across the text to push characters aside like ripples in water. All displacement, rotation, scaling, and return animation is pure CSS transforms and transitions.

Inspired by [@ohhdanqi](https://x.com/ohhdanqi).

**[Live Demo](https://zebiv.com/text-ripple-css/)**

## Tech

- Vanilla HTML/CSS/JavaScript
- CSS transforms and transitions for all visual effects
- JS only for mouse tracking and distance calculation
- EB Garamond serif typography
